# Módulo Cliente - IURIS
Este directorio contiene la documentación técnica y diagramas UML correspondientes a las funcionalidades y comportamientos del actor "Cliente" dentro del sistema IURIS.

## Estructura de Documentación
- `1-Casos-Uso/`: Definición de requisitos funcionales.
- `2-Secuencias/`: Interacción paso a paso entre objetos.
- `3-Actividades/`: Flujos de procesos de negocio.
- `4-Estado/`: Ciclo de vida de objetos principales.
- `5-Tiempos/`: Diagramas de temporización (Time diagrams).
- `6-comunicacion/`: Relaciones estructurales entre componentes.
- `7-Diagrama_estados/`: Máquina de estados detallada.
